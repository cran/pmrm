Validation

We run a simulation study to verify the models in pmrm are implemented correctly. For each simulation scenario (model type and disease progression trajectory), we:

  1. Simulate 1000 datasets from the model using independent non-RTMB R code.
  2. Fit the model to each simulated dataset using the RTMB-powered implementation.

Around half of the 50% confidence intervals from (2) should cover the true data-generating parameter values in (1). Likewise, around 95% of the 95% confidence intervals from (2) should cover the truth. In addition, we expect all models to converge.

This simulation study is a targets pipeline in the vignettes/validation/ directory of the source code of pmrm. To run the pipeline, change to that working directory and call targets::tar_make(). Due to the computational cost, we only run this pipeline once per release.
